Does ESG controversy add signal?

Waiting for ESG model comparison.

Nested feature comparison: the baseline uses only price/fundamental/downside risk features; the full model adds eight ESG controversy variables (score level, 4/13/26-week changes, rolling mean & std, spike flag, sector percentile).

Econometric read: the lift rows report the out-of-sample improvement from adding ESG controversy data on the same chronological train/validation/test splits.

ROC-AUC ranks risky names above safer names. Precision@Top Bucket asks how many flagged high-risk names actually become high-risk. Crash Capture asks how many future high-risk names were caught in the top bucket.
